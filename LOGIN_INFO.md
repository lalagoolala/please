# 🔐 로그인 정보

계정을 만들거나 구글 계정을 사용하여 로그인할 수 있습니다.

---

## 🚀 Google 로그인 (권장)

"Google로 로그인" 버튼을 클릭하여 별도의 가입 절차 없이 편리하게 로그인할 수 있습니다.

### Google 로그인 설정 안내
Google 로그인을 실제 서비스에서 활성화하려면 다음 단계가 필요합니다:
1. [Google Cloud Console](https://console.cloud.google.com/) 접속
2. 새 프로젝트 생성 및 OAuth 동의 화면 구성
3. OAuth 2.0 클라이언트 ID 생성
4. `index.html` 소스 코드의 `YOUR_GOOGLE_CLIENT_ID` 부분을 실제 발급받은 ID로 변경

---

## 📝 일반 회원가입 및 로그인

### 회원가입하기
1. "📝 회원가입" 버튼 클릭
2. 이름, 이메일, 계정 유형(학생/학부모/관리자) 입력
3. 가입 후 관리자의 승인을 기다려주세요.

### 로그인하기
1. "🔐 로그인" 버튼 클릭
2. 아이디(이메일 ID)와 비밀번호 입력

---

## 🎯 테스트 방법

1. **Google 로그인:** 버튼이 정상적으로 표시되는지 확인합니다. (클라이언트 ID 설정 전까지는 오류 메시지가 뜰 수 있습니다.)
2. **일반 회원가입:** 정보를 입력하여 가입 신청을 합니다.
3. **관리자 승인:** 현재 시스템은 로컬 스토리지를 사용합니다. 테스트를 위해 가입 즉시 승인된 상태로 전환하거나 다른 브라우저 세션을 확인하세요.

---

## 💡 참고 사항

- 이 홈페이지는 브라우저의 로컬 스토리지(LocalStorage)를 사용하여 데이터를 저장합니다.
- 브라우저 쿠키나 캐시를 삭제하면 저장된 회원 정보가 초기화될 수 있습니다.
